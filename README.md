## üöÄ Getting Started

1. [Fork](https://github.com/Orloxx23/time-sync/fork) or clone this repository.

```bash
git clone git@github.com:Orloxx23/time-sync.git
```

2. Install the dependencies:

```bash
npm install 
```

3. Run the development server:

```bash
npm run dev
```

4. Open [**http://localhost:3000**](http://localhost:3000/) with your browser to see the result üöÄ
